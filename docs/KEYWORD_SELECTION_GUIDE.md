# キーワード選定ガイド

シリーズ企画のキーワードは、企画の方向性を決める重要な入力です。
このガイドでは、良いキーワード選定のコツを説明します。

## 基本原則

### 1. 具体的な要素を含める
悪い例: `ファンタジー 冒険 魔法`
良い例: `異世界転生 魔法使い 王都 陰謀 成長`

### 2. 対立構造を示唆する
- `主人公 vs 敵対者`
- `理想 vs 現実`
- `義務 vs 欲望`

### 3. ジャンルの核心を伝える
| ジャンル | 含めるべき要素 |
|---|---|
| ファンタジー | 魔法体系、種族、世界のルール、古代文明 |
| SF | テクノロジー、社会構造、倫理的問題 |
| ミステリー | 犯人、動機、トリック、探偵 |
| ロマンス | 障壁、誤解、成長、決断 |
| 歴史 | 時代背景、実在人物、架空の交差 |

## 良いキーワード例

### 例 1: 異世界転生ファンタジー
```
異世界転生 魔法使い 王都 陰謀 成長 契約魔法
```

### 例 2: 近未来SFミステリー
```
近未来 記憶操作 探偵 企業陰謀 アンドロイド 記憶の真実
```

### 例 3: 歴史ロマンス
```
幕末 志士 女性剣士 恋愛 開国 派閥争い
```

### 例 4: 現代ミステリー
```
地方都市 失踪 過去の事件 記憶喪失 警察 内部告発
```

### 例 5: 学園青春
```
高校生 部活動 幼馴染 三角関係 進路 秘密
```

## 避けるべきキーワード

| 避ける表現 | 理由 |
|---|---|
| `面白い物語` | 主観的すぎる |
| `感動する` | 目的であって素材ではない |
| `人気のジャンル` | 具体性がない |
| `王道` | どの王道か不明 |
| `ユニークな設定` | 具体性がない |

## キーワードの構成テンプレート

```
[ジャンル] [主人公の立場] [舞台] [核心的な対立] [テーマ/キーワード]
```

例:
```
近未来SF 記憶捜査官 刑事 サイバーパンク 記憶の真実
```

## よくある失敗と修正

| 失敗例 | 修正案 |
|---|---|
| `恋愛 ファンタジー` | `異世界転生 令嬢 契約結婚 魔法 王子` |
| `SF アクション` | `近未来 サイバネティクス 傭兵 企業戦争 AI` |
| `ホラー ミステリー` | `閉鎖空間 連続殺人 過去の呪い 生存 サスペンス` |
