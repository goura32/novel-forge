# NovelForge キーワード選定ガイド

シリーズ企画（`plan` 工程）の入り口となるキーワードは、作品の品質を決定する最も重要な入力です。
ここでは具体的なキーワードの選び方を選別基準・役割・チェックリスト付きで定義します。

---

## 基本方針

> **キーワードは、「雰囲気語」ではなく「物語を動かす具体語」を選ぶ。**

LLM は抽象的な雰囲気語を理解しません。具体的に何が起こるかを指示する言葉を与えて初めて、
一貫した筋書きが生成できます。

**具体的な入力は具体的で面白い出力を生む。** それだけのことです。

---

## 8役割キーワードの選定

1セットとして **8つの役割** でキーワードを構成します。

| No. | 役割 | 内容 | 補足 |
|:---:|------|------|------|
| 1 | **主人公の職能** | 主人公の職業・持ちスキル・専門性 | 「探偵」より「保険調査員」「元看護師」が具体的 |
| 2 | **恋愛相手の職能** | 相手側の職業・属性 | 同じ業界、ライバル業種、上下関係など設定に幅を持たせる |
| 3 | **舞台** | 物語の場所・舞台設定 | 都道府県、施設名、職場環境など具体的な地点 |
| 4 | **謎の物証** | 物語を動かす物理的な物体・証拠 | データファイル、遺品、記録、写真、メールなど |
| 5 | **違和感・調査のきっかけ** | ストーリー発端の出来事 | 見慣れないもの、想定外の発見、日常のずれ |
| 6 | **二人の関係性** | 主人公と相手の初めの関係 | 同僚・取引先・ライバル・元恋人・上司部下 など |
| 7 | **生活・仕事上の障害** | プラスαの困難要素 | タイムリミット、予算制約、人間関係、権力構造 |
| 8 | **物語後半のイベント** | 中盤で起こる転機（日常レベル） | 突然の異動、事件の発覚、組織の混乱など |

### 選び方のコツ

- 各役割が「具体的に何を示すか」が明確になる言葉を選ぶ
- 「どの職業の人が・どこで・何をきっかけに・どう動くか」が読めるようにする
- 8つ全部が独立して機能する（組み合わせによってストーリーが膨らむ）ことが理想

---

## 避ける語（原則禁用）

以下の単語は **設定の陳腐化** や **プロットのパターン化** を招くため、原則避けます。

### パッと読んでジャンルがわかる用語
- 悪役令嬢、婚約破棄、聖女、公爵、王太子
- 溺愛、ざまぁ、追放

### 異世界転生関連
- 前世、ループ、死に戻り

### 抽象的すぎる感情・テーマ語
- 記憶、思い出、運命、奇跡、真実、秘密
- 静かな恋、遅い恋、新しい居場所、心の回復

### パターン化しやすいフレーズ
- 「古い〇〇」の多用
- 「消えた〇〇」の多用

## 良いキーワードの条件

選択するkeywordが以下条件を **1つ以上** 満たしている場合、良いキーワードといえます。

| # | 条件 | 具体例 |
|:---|------|--------|
| ① | **表紙やタイトルに使える** | 「隠密」「記録」「工場」→ 視覚的な連想が働きやすい |
| ② | **あらすじに自然に入る** | 文脈をつなげる必要なく、そのまま一文能成为する |
| ③ | **主人公が行動する理由になる** | その言葉で「なぜ動くか」が説明つく |
| ④ | **恋愛相手と協力する理由になる** | 二人が一緒に動く必然性が生まれる |
| ⑤ | **軽いミステリーの入口になる** | 「何かがおかしい」というフックに直結する |
| ⑥ | **大事件ではなく、日常の違和感から始められる** | 普通の人々が普通の場所で小さな異変に気づく形 |

条件を1つでもクリアしていればOK。複数が重なるとさらに良いです。

---

## チェックリスト

キーワードセットを作成したら、以下のチェックを行います。

- [ ] 各役割が「具体的なもの」になっているか（場面が頭に浮かぶか）
- [ ] 8個すべてに独立した意味があるか
- [ ] 禁用語が含まれていないか
- [ ] おまけ: アウトライン生成時にそのまま使えるキーワードなのか

---

## 比較例

### NG: 抽象的・雰囲気重視
```
異世界, 令嬢, 転生, 婚約, 魔法
```
→ 何をすればよいのかわからない。プロンプトが空回りする。

### OK: 具体語で物語を動かす
```
保険調査員, データ保管業者, 倉庫街, 消えたデータファイル
, 顧客情報の不具合, 元同僚, 倒産回避, 隠蔽体質の企業
```
→ 主人公の職業が明確。舞台も具体語。謎の物証・違和感のきっかけがある。調査が自然に始まる。
