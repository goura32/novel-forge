# シーンの改稿

## 指示

以下のシーン本文を、レビュー結果に基づいて改稿せよ。新規生成ではない。

## 入力

- シーン本文: `{scene}`
- レビュー結果: `{review}`
- 出力言語: `{lang}`

## 改稿方針

1. `critical` の issue は必ず修正すること
2. `major` の issue は可能な限り修正すること
3. `minor` の issue は余力があれば修正すること
4. 原文の良さを活かしつつ、問題点を改善すること
5. レビューで指摘されていない部分を勝手に変更しないこと
6. `kanji_issues` に列挙されている漢字（簡体字等）が本文中にあれば、日本語の正しい漢字に置換すること

## 出力形式

Markdown 形式の改稿本文を出力すること。JSON ではない。
