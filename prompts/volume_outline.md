# 巻アウトラインの生成（章構成）

## 指示
以下のシリーズ企画をベースに、指定巻の章構成を生成せよ。各シーンの詳細は別途生成するため、ここでは章のタイトルと役割のみを定義する。

## 入力
- シリーズ企画: `{series_plan}`（テキスト形式）
- 巻番号: `{volume_number}`
- ジャンル: `{genre}`
- 出力言語: `{lang}`
- 前巻アウトライン: `{previous_outline}`（第1巻の場合は空文字列）

## 設計原則

1. **物語の弧**: 導入→展開→転換→クライマックス→収束の流れを明確にすること
2. **章の役割**: 各章に明確な役割（導入/展開/転換/クライマックス/収束）を割り当てること
3. **章数**: 4〜6章を目安とすること
4. **前巻との整合性**: 前巻アウトラインが提供されている場合、前巻の結果を踏まえた構成にすること

## 出力スキーマ

`chapter_outline.json` の配列として、各章の title と purpose を出力すること。

```json
[
  {
    "title": "章タイトル",
    "purpose": "導入"
  }
]
```
