# シリーズ企画の核（コア）の生成

## 指示
以下のキーワードから、商業出版を意識した小説シリーズの企画核を生成せよ。

## 入力
- キーワード: `{keywords}`
- 出力言語: `{lang}`

## 企画の要件

1. **市場意識**: 競合他社との差別化ポイントを明確にすること
2. **ターゲット読者**: 年齢層・性別・好みを具体的に定義すること
3. **タイトル**: 検索力・宣伝力を考慮したタイトル案を提示すること
4. **世界観**: 独自のルールを持つ世界観を定義すること

## 出力スキーマ
`series_plan_core.json` に適合する JSON を出力すること。

```json
{
  "title": "シリーズタイトル",
  "logline": "あらすじ（1-2文）",
  "genre": ["ジャンル1"],
  "target_audience": "ターゲット読者",
  "themes": ["テーマ1"],
  "selling_points": ["売りポイント1"],
  "world": {
    "summary": "世界観の概要",
    "rules": ["ルール1"]
  }
}
```

**注意**:
- `genre` は1-3個
- `themes` は1個以上
- `selling_points` は1個以上
- `world.rules` は1個以上
- JSONの文字列値内に改行を含めないこと

言語: {lang}
