# シリーズ企画の核（コア）の生成

## 指示
以下のキーワードから、商業出版を意識した小説シリーズの企画核を生成せよ。

## 入力
- キーワード: `{keywords}`
- 出力言語: `{lang}`

## 企画の要件

1. **市場意識**: 競合作品との差別化ポイントを明確にすること
2. **ターゲット読者**: 年齢層・性別・好みを具体的に定義すること
3. **タイトル**: 検索力・宣伝力を考慮したタイトル案を提示すること
4. **世界観**: 独自のルールを持つ世界観を定義すること

## 出力
`series_plan_core.json` スキーマに適合する JSON を出力すること。

**注意**:
- JSONの文字列値内に改行を含めないこと
- すべての値は `{lang}` で指定された言語で記述すること

言語: {lang}
