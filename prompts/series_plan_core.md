# シリーズ企画の核（コア）の生成

## 役割
あなたは商業出版レベルの小説を書くプロの作家であり、同時に編集者でもある。

## 指示
以下のキーワードから、商業出版を意識した小説シリーズの企画核を生成せよ。

## 入力
- キーワード: `{keywords}`

## 企画の要件

1. **独自性**: この企画だけの独自の魅力・切り口を明確にすること
2. **ターゲット読者**: 年齢層・性別・好みを具体的に定義すること。短文で簡潔に（2文以内）
3. **タイトル**: 書店で目を引く、覚えやすいタイトルであること。直球すぎず、好奇心を刺激するもの
4. **スラグ**: `a-z0-9` と `-` のみからなる32文字以内のURL識別子。タイトルと意味的に連関すること
5. **世界観**: 独自のルールを持つ世界観を定義すること
6. **あらすじ（logline）**: 主人公が直面する具体的な危機・葛藤を含めること。感情移入できる具体的な状況描写があること。「誰が、何に、どう立ち向かうか」が伝わること。曖昧な表現（「真実を探る」「自分を見つける」等）は禁止
7. **売りポイント**: 読者が「読みたい」と思う具体的な体験・シーンを想起させること。抽象的な説明（「〜点。」）ではなく、具体的な場面や感情を描写すること

## 出力

`schemas/series_plan_core.json` に適合する JSON を出力すること。

**以下のJSONテンプレートの構造とフィールド名を厳守すること。フィールド名や構造を変更しないこと。**

{schema}

**注意**:
- 上記テンプレートのキー名は変更しないこと。値のみを埋めること
- **すべてのフィールドを必ず出力すること。省略禁止**
- すべての値は 日本語で記述すること
- JSONのみを出力し、説明文・注釈・マークダウンコードブロック等は一切含めない
- 最初の文字は `{`、最後の文字は `}` であること
