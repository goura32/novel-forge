# 設定資料集（Bible）の更新

## 指示

以下のシーン本文から、設定資料集（Bible）に反映すべき情報を抽出せよ。

## 入力

- シーン本文: `{scene}`
- 現在のBible: `{bible}`
- 出力言語: `{lang}`

## 抽出項目

1. **新キャラクター**: 新しく登場したキャラクター情報を `characters` に追加
2. **新用語**: 新しく使用された用語を `glossary` に追加
3. **伏線**: 設置された伏線を `foreshadowing` に追加
4. **伏線回収**: 回収された伏線の `resolved` を `true` に更新
5. **世界観ルール**: 明示された世界観ルールを `world_rules` に追加

## 出力スキーマ

`bible.json` に適合する JSON を出力すること。
