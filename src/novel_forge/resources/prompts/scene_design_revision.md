## 重要: 出力形式

**以下の8つのフィールドを含む JSON オブジェクトのみを出力してください。他のフィールドは絶対に含めないでください。**

```json
{
  "title": "シーンタイトル（文字列）",
  "pov": "視点キャラクター（文字列）",
  "goal": "シーンの目標（文字列）",
  "conflict": "シーンの葛藤（文字列）",
  "outcome": "シーンの結果（文字列）",
  "characters": ["キャラクター1", "キャラクター2"],
  "key_events": ["イベント1", "イベント2"],
  "setting": "舞台設定（文字列）"
}
```

**各フィールドは必ず文字列（または配列）で指定してください。null や undefined は絶対に禁止。**
