# シーン本文改訂

## 目的
レビューで指摘された本文上の問題を投影済み writer context の範囲で最小限修正し、完成版を返す。

## 応答方針
`issues[].field` に関係しないフィールドは原則として元の値を保持する。整合性調整が必要な場合だけ、最小限変更する。明示的な指摘がない限り変更しない。本文外の事実や POV が観測できない内面を追加しない。

## 実行指示
スキーマどおり `title` と `content` を返す。`title` に対する issue がなければ、元の値をそのまま保持する。`content` には、指摘された問題を反映した完成した日本語本文を入れる。

改訂では `writer context` を事実の正本として再確認する。review が指摘した現在状態・POV の知覚・人物の接触可能性・環境変化の範囲・アイテム機能を、context の明示内容に照らして本文から矛盾なく直す。context にない能力や設定を補って解決しない。特に、知覚制限を守るために未定義の感官を足さず、局所的な異変を全体的な環境変化へ拡大せず、明示されない道具の戦闘・増幅・反射機能を追加しない。

「特殊な装置が存在しない」という制約は装置の使用・示唆を除くものであり、context で許可された人物自身の五感・経験・直感を消す根拠にはしない。装置ではない観測は、観測者の感覚として明確に書き分ける。
