# シーン本文の生成

## 目的
投影済み writer context と直前 continuity handoff に忠実な完成日本語小説本文を生成する。

## 応答方針
Canon、作者だけの真相、stable ID を推測しない。POV が観測できる事実だけを使い、現在進行の行動・異変・欲求から始める。

## 実行指示
約500〜5000字の自然な日本語本文を `content` に書く。説明・見出し・メタ注釈を本文に混ぜない。

### writer context 制約の厳守
`writer context` は本文生成の唯一の正本である。`current_state` / `cast_constraints` / `setting_state` / `setting_constraints` / `observable_state` に書かれた内容を、このシーン開始時点の確定事実として扱う。雰囲気づけや展開のために、それらと反対の状態・位置・関係性・能力を推測で足さない。

本文を書く前に、POV が今この場面で観測できる感官、登場人物の現在地と接触可能性、場所・環境変化の有効範囲、アイテムが明示的に持つ機能を `writer context` から読み取る。本文 `content` はこの範囲だけで描写する。特に、局所的な変化を室内全体へ広げない、明示されない戦闘・増幅・反射等の道具機能を作らない、解消済みまたは離脱中の人物を未解消・近接中として扱わない。

制約に「特殊な装置が存在しない」とある場合、その装置を使う・示唆する描写を書かない。ただし、同じ context で許可される人物の五感・経験・直感による観測まで装置のように扱わず、本文では観測者自身の感覚として区別して描く。

具体的には以下を破る描写を書かないこと：
- 結界の範囲（例：店外では妖怪の力に直接干渉できない。店内からの観測は「気配・音・漏れ光」程度に留め、店外の妖怪へ直接干渉・防衛行為は書かない）
- 知覚の限界（例：店主以外の人間には妖怪が見えない、または存在を誤魔化された状態で見えている。POV が通常観測できないものを明確に視認・観察する描写は書かない）
- 年代の統一（例：明治初期＝1870年代。別の時代表記と混在させない）
- 物理法則の扱い（例：怪異の影響であっても急激な極端な環境変化は避ける）
- Canon で定義されていない怪異名・術名・固有名詞を新規に導入しない（既存 entity の名前のみ使用）

制約に反する描写は、後続工程で整合性が固定化されるため書かないこと。
